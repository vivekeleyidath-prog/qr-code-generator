# QR Code Generator

A Python desktop application that generates QR codes from text or URLs.

## Requirements

- Python 3.x
- qrcode library

## Installation

Install the required package:

```bash
pip install qrcode[pil]
```

## Run the Application

```bash
python GenQr.py
```
